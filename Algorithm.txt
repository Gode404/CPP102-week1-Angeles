ALGORITHM

STEP 2: Ask the user to enter the first number. Store it as num1.

Laboratory Activity 2: Compute the Average of Three Quiz Scores
ALGORITHM

STEP 1: START 
STEP 2: Ask the user to enter the first number. Store it as score1.
STEP 3: Ask the user to enter the second number. Store it as score2.

Laboratory Activity 3: Determine if the Number is Even or Odd
ALGORITHM 

STEP 1: START. 
STEP 2: Ask the user to enter a number. store is as number.
STEP 3: Check if number mod 2 = 0.
STEP 4: If true, display "The number is Even."
STEP 5: If not, display "The number is Odd."
STEP 6: End.
